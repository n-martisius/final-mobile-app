Final Mobile App (Expo + React Native)

This is a mini mobile application built with React Native, Expo, and Expo Router as part of the final graded assignment. The app demonstrates navigation, local storage, API data fetching, and a clean dark-themed UI.


How to Run the App

1. Install dependencies:

npm install


2. Start the Expo development server:

npm start


3. Run the app using:

Expo Go on a physical device or Web browser


API Used

The application uses the JSONPlaceholder Users API:

https://jsonplaceholder.typicode.com/users


Data displayed from the API:

- User name

- User email


Features

- Multi-screen navigation using Expo Router

- Profile storage using AsyncStorage (name and city)

- API list screen with data fetched using fetch()

- Details screen with navigation parameter

- Search/filter functionality in the API list

- Pull-to-refresh support in the API list

- Dark theme UI with readable layout and spacing

- Basic error handling and loading indicators
